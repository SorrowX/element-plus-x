<p align="center">基于 Element Plus 拓展的组件库</p>

- 💪 Vue 3 Composition API
- 🔥 Written in TypeScript

## 安装

node 版本: 18.19.1; npm 版本: 10.2.4; pnpm 版本: 9.3.0

```js
pnpm install
```

## 本地调试

```js
npm run dev
```

## 创建新组件

```js
// test为组件名
npm run gen test
```

## 源代码构建

```js
npm run build
```

## 文档调试

```js
npm run docs:dev
```

## 文档构建

```js
npm run docs:build
```
